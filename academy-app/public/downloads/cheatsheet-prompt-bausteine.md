# Cheatsheet: Prompt-Bausteine

Nachschlagen statt raten: Was jeder der 5 Bausteine (Rolle · Kontext · Aufgabe · Format · Beispiel) bewirkt – mit gutem und schlechtem Beispiel im direkten Vergleich. Neben die Tastatur legen, bis es sitzt.

## 1. Rolle

**Was sie bewirkt:** Die Rolle bestimmt Wortwahl, Fachniveau und Perspektive der Antwort. „Erkläre wie ein Steuerberater" und „erkläre wie ein Lehrer für die 8. Klasse" liefern zum selben Thema völlig verschiedene Texte.

| | Formulierung |
|---|---|
| ❌ Schlecht | „Du bist ein Experte." (Experte wofür? Sagt der KI nichts.) |
| ✅ Gut | „Du bist HR-Verantwortliche in einem Schweizer KMU mit 40 Mitarbeitenden und schreibst oft Stelleninserate." |

## 2. Kontext

**Was er bewirkt:** Kontext ersetzt Raten durch Wissen. Ohne Kontext füllt die KI die Lücken mit dem Durchschnitt des Internets – und der passt selten zu deinem Fall.

| | Formulierung |
|---|---|
| ❌ Schlecht | „Schreibe eine Absage an einen Bewerber." |
| ✅ Gut | „Schreibe eine Absage an einen Bewerber, der in der zweiten Runde war und fachlich stark, aber teurer als das Budget war. Wir wollen ihn in 6 Monaten evtl. nochmal kontaktieren." |

## 3. Aufgabe

**Was sie bewirkt:** Die Aufgabe ist der Auftrag – ein Verb, ein Ziel, ein Umfang. Je prüfbarer, desto besser das Ergebnis.

| | Formulierung |
|---|---|
| ❌ Schlecht | „Kannst du mir mit meiner Präsentation helfen?" (Hilfe womit – Gliederung? Text? Kürzen?) |
| ✅ Gut | „Kürze die folgenden Sprechernotizen auf max. 3 Sätze pro Folie, ohne Zahlen zu verändern." |

## 4. Format

**Was es bewirkt:** Das Format macht die Antwort direkt weiterverwendbar – als Tabelle für Excel, als Stichpunkte für Slack, als Mail zum Kopieren. Ohne Formatangabe bekommst du Fließtext-Absätze, die du erst umbauen musst.

| | Formulierung |
|---|---|
| ❌ Schlecht | „Mach es übersichtlich." |
| ✅ Gut | „Tabelle mit den Spalten Aufgabe \| Verantwortlich \| Frist. Danach maximal 3 offene Fragen als Liste." |

## 5. Beispiel

**Was es bewirkt:** Ein Beispiel zeigt, was du meinst, statt es zu beschreiben. Besonders bei Tonalität schlägt ein einziger Beispielsatz jede Adjektiv-Liste.

| | Formulierung |
|---|---|
| ❌ Schlecht | „Schreib professionell, aber nicht steif, modern, auf Augenhöhe." |
| ✅ Gut | „Schreib im Ton dieses Satzes aus unserem letzten Newsletter: ‚Wir halten es kurz: Ab Januar ändern sich drei Dinge für Ihre Buchhaltung.'" |

## 8 Verstärker-Snippets zum Wiederverwenden

Diese Sätze kannst du an fast jeden Prompt anhängen. Kopieren, einfügen, fertig:

```
Stelle mir Rückfragen, bevor du antwortest, falls dir wichtige Informationen fehlen.
```
```
Antworte in Stichpunkten, maximal 10.
```
```
Gib mir 3 Varianten: eine sichere, eine mutige, eine kurze.
```
```
Wenn du dir bei etwas nicht sicher bist, markiere es mit [UNSICHER] statt zu raten.
```
```
Erkläre danach in einem Satz, warum du dich so entschieden hast.
```
```
Verwende ausschließlich Informationen aus dem Text, den ich dir gebe – erfinde nichts dazu.
```
```
Kritisiere zuerst meinen Entwurf schonungslos, verbessere ihn erst danach.
```
```
Schreibe auf Deutsch (Schweiz): ss statt ß, CHF statt €, Datumsformat 15.07.2026.
```

## Faustregeln

- [ ] Kurze Alltagsfrage → Aufgabe allein reicht.
- [ ] Ergebnis geht an Kunden oder Chef → alle 5 Bausteine.
- [ ] Ton stimmt nicht → Beispiel nachliefern.
- [ ] Antwort zu allgemein → Kontext nachliefern.
- [ ] Antwort unbrauchbar strukturiert → Format nachliefern.
